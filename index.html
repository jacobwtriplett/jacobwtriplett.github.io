stackdid: Stacked difference-in-differences regression

Remarkably straightforward stacked difference-in-differences regressions. An assortment of options enable precise model specification and robustness analysis.

gintreg: Generalized interval regression

Estimate linear models with possibly nonnormal disturbances and with outcomes measured as point, interval, left-censored, or right-censored data. Specify any of seventeen distributions in the Skewed Generalized t (SGT) family and Generalized Beta of the Second Kind (GB2) family.

GitHub Repository → Paper →

goprobit2: Generalized ordered probit regression

Fit ordered response regression models of ordinal outcomes. goprobit2 extends oprobit by offering a variety of link functions based on the Skewed Generalized t (SGT) family of statistical distributions.

GitHub Repository →
